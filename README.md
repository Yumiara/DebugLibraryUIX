# Debug Library Document

## Module
```lua
    local Module = loadstring(game:HttpGet("https://raw.githubusercontent.com/Yumiara/DebugLibraryUIX/main/Library.lua"))();
```

## Acrylic Library
### Create Acrylic
```lua
    Module["Acrylic"].new(<Frame>);
```

### Clear Acrylic
```lua
    Module["ClearAcrylic"]();
```
